# ✅ ToDoApp

A clean and responsive To-Do application built with React. Manage your daily tasks with a simple and intuitive interface.

🔗 **Live Demo:** [to-do-app-gp65.vercel.app](https://to-do-app-gp65.vercel.app/)

---

## 🚀 Features

- Add, complete, and delete tasks
- Responsive design — works on desktop and mobile
- Clean UI with custom SCSS styling
- Fast and lightweight — built with React

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| React | UI framework |
| JavaScript (ES6+) | Application logic |
| SCSS / CSS | Styling |
| Create React App | Project bootstrapping |
| Vercel | Deployment |

## 📦 Getting Started

### Prerequisites

- Node.js (v14+)
- npm

### Installation

```bash
git clone https://github.com/RnUrPl/ToDoApp.git
cd ToDoApp
npm install
```

### Running locally

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for production

```bash
npm run build
```
